# San-Jose-Employee-Compensation-Analysis

For this project, I analyzed compensation data from the City of San José to understand salary and overtime trends, department-level employee spending, and pay disparities. 

Using PostgreSQL, I performed exploratory data analysis (EDA) to identify key trends. Then, I built an interactive [Tableau dashboard](https://public.tableau.com/shared/Y3CKTSX7X?:display_count=n&:origin=viz_share_link) to visualize key insights for decision-makers. 

The question that this analysis answered is as follows:
1. What's the average base pay/salary?
2. Minimum and Maximum salary by department
3. 10 highest overtime jobs?
4. Which departments are the most expensive?
5. Which departments rely the most on overtime?
6. The top 10% of earners by department
7. Calculating overtime as a % of total compensation.
